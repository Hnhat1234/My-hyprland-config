### Installation

You need to install hyprland first. And then the additonal packages.

Packages:

* wofi
* zenity
* xdg-desktop-portal
* xdg-desktop-portal-hyprland
* waybar
* hyprpolkitagent
* hyprlauncher
* kitty
* thunar

Running the git clone in the ```~/.config/``` directory:

```bash
git clone https://github.com/Hnhat1234/My-hyprland-config.git
```

In the hyprpaper.config, remmeber to change your monitor, and wallpaper:

```hyprland
wallpaper {
    monitor = <Your monitors>
    path = /path/to/wallpaper.png
    fit_mode = <Some mode>
}
```
